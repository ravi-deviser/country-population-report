# Country Population Report

Python project that fetches country population data from a public API.

## Features
- Fetches live country data
- Groups data by region
- Calculates total population
- Saves output in Excel, CSV, and JSON

## Tech Stack
- Python
- requests
- openpyxl

## How to Run
pip install requests openpyxl
python country_population_report_level_1.py
